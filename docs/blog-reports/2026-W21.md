## How to intervene

- **Kill a bad topic:** edit `docs/blog-queue.json`, set the topic's status to `rejected` and add a `rejectionReason`.
- **Force a specific topic next:** reorder `docs/blog-queue.json`; the draft agent picks highest-scored `new`/`scheduled` entry.
- **Pause the pipeline:** disable the workflow in GitHub Actions UI (Settings → Actions → blog-pipeline).
- **Adjust voice:** edit `docs/blog-voice-examples.md` ,  the writer agent picks it up on the next run.
- **Adjust style rules:** edit `docs/blog-style-guide.md`.
- **Adjust budget cap:** edit `WEEKLY_CAP_USD` in `scripts/blog/lib/budget.mjs`.
